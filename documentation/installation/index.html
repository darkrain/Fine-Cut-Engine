Fine Cut is an archive bundle. And it have no any installation script, becaue it made simple to use.

Only thing you need to install and use Fine Cut is to unpack bundle to any directory of the website, which is availiable for browsing.

We suppose it will be Apache webserver, however, we also suppose it could be used in another sever, but there is no config for other servers now. So, if you'll use another type of webserver, you'll need to look inside .htaccess file to make proper setup for using Fine Cut.


So, again, step by step:

  1. Download relevant version of Fine Cut: ZIP or TAR.BZ
  2. Choose folder where you wish to install it.
  3. Unpack downloaded archive there.Please, note that there is folder inside that bundles. One folder deep inside ZIP, and two folder deep inside TAR.BZ. You need to copy files from there to choosed folder after unpacking.
  4. Go to that directory URL through browser after unpacking and copying, you shoud look at index page, if not, then, something gone wrong, may be mod_rewrite is switched off for your Apache.
  5. Go inside «_adm» dir. This is an Admin Panel.

That is all, Fine Cut is independent of where it is installed in.

Default template and pages of Fine Cut are exactly the same you are looking here.


So, this is all about how to install Fine Cut Engine.
