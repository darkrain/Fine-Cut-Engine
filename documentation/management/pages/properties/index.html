We use this page to create a hierarchy structure of page's necessary data.

It could be used to store, for example, goods properties, or menu, or something else with hierarchical direction.


Please, remember that here is strange behaviour.

When you will write something inside of a tree block, DO NOT SAVE PAGE IMMEDIATELY!

At first you need to STEP OUT of written chunk, because otherwise data will not be remembered inside of the hierarchy model.


For example, there is «Title» block exactly below textarea.

This block contains keyword «"Title"».

If you will step inside of block with «"Title"» keyword, then edit it, then, before saving page you'll need to step out from editing mode, inspite of loosing data. Editing mode switches off only when you step out of it. To step out just click page to an empty space. And, while editing mode is not switched off, data will not be remembered.
