Here you could see we use a brilliant ACE Editor.


Only small addition we made are hotkeys preset:

  • Ctrl + 1, Ctrl + 2 ... Ctrl + 6 : for Headings <H1>, <H2> ... <H6>

  • Ctrl + P : for new paragraph <P>

  • Ctrl + B, Ctrl + I : <STRONG> bold and <EM> italic

  • Ctrl + L : for new link <A HREF=">

  • Ctrl + Q : for &laquo;|&raquo; – «|»

  • Ctrl + 7 : for the dash &ndash; –

  • Ctrl + 8 : for the long dash &mdash; —

  • Ctrl + 9 : for new &nbsp; –                   *here we are with using it.

  • Ctrl + S : for saving page without mouse usage...

    And You could use Ctrl + S on every tab of «Pages Manager» also...
