<p>Fin Cut is bundled as an archive.</p>
<br />

<p>It don't need any DB, therefore installation process is extremely simple.<br /><br />

<strong>Just unpack Fincut Archive to web-root folder or any folder inside.</strong></p>
<br />

<p>But, first of all, please, read <a href="/documentation/requirements/" target = "_blank">Requirements</a>.</p>

<p>Then, if all done simple, You could go to http://YourSite/YourInstallDir/_adm.</p>
<p>This is admin panel.</p>

<p>Please read more <a href="/documentation/installation/">inside</a> of the documentatoin.</p>
