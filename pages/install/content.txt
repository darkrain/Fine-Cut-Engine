<p>Fincut is bundled as an archive.</p>
<br />

<p>It don't use any DB, therefore installation process is extremely simple.<br />
<strong>Just unpack Fincut Archive to web-root folder or any folder inside.</strong></p>
<br />

<p>Then, if all done simple You could go to http://YourSite/YourInstallDir/_adm.</p>
<p>This is admin panel.</p>
<p>One only thing, if you will not use sub directory {YourInstallDir}, then you'll to move inside settings and put working directory preference there: $prepath. There You'll find examples how to do this.</p>
<br />

<p>Please read more <a href="/documentation/installation/">inside documentatoin</a>.</p>
